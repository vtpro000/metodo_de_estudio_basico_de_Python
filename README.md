Ejercicio: Sistema Bancario Colaborativo en Python

Una entidad financiera llamada Banco Orion necesita desarrollar un sistema bancario básico en Python para modernizar la atención a sus clientes.
El proyecto será realizado por un equipo de desarrolladores, por lo que cada integrante deberá trabajar en módulos separados, usando funciones, archivos e importaciones para unir todo el sistema al final.

El objetivo es crear una aplicación en consola que permita administrar cuentas bancarias de manera simple, segura y ordenada.

Requerimientos del sistema

El sistema debe permitir:

Registrar nuevos clientes.
Crear cuentas bancarias.
Iniciar sesión con número de cuenta y clave.
Consultar saldo disponible.
Depositar dinero.
Retirar dinero verificando fondos suficientes.
Transferir dinero entre cuentas.
Mostrar historial de movimientos.
Guardar información mientras el programa esté funcionando.
Tener un menú principal interactivo.
Reglas del sistema
No se pueden ingresar montos negativos.
No se puede retirar más dinero del disponible.
Cada cuenta debe tener un número único.
Las contraseñas deben validarse antes de ingresar.
Cada movimiento debe quedar registrado.
El programa debe manejar errores para evitar que se cierre inesperadamente.

Objetivo Final

El equipo deberá entregar un sistema bancario funcional en consola capaz de:

Registrar usuarios.
Iniciar sesión.
Realizar operaciones bancarias.
Mostrar historiales.
Trabajar mediante módulos conectados.

El sistema debe funcionar correctamente aunque cada integrante trabaje por separado, simulando un entorno real de desarrollo colaborativo en programación.
